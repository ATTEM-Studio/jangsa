## 현재 데이터 처리

- 매장 입력값과 진단 이력은 사용자의 브라우저 `localStorage`에만 저장됩니다.
- 현재 버전은 서버로 매출·광고 데이터를 전송하지 않습니다.
- URL에는 진단 원본 데이터를 넣지 않습니다.
