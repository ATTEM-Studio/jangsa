# AGENTS.md

## 제품 목표

장사네비게이션은 외식업 자영업자가 10초 안에 오늘 할 행동 하나를 이해하고 30분 안에 실행하도록 돕는 도구다.

## 반드시 지킬 원칙

- 결과 첫 화면에는 가장 중요한 행동 하나만 보여준다.
- 모든 추천은 실행 가능 조건을 먼저 확인한다.
- 데이터가 부족하면 추정으로 단정하지 말고 확인할 숫자를 요청한다.
- 예상효과는 보수적인 범위와 계산 근거를 함께 제공한다.
- 고객 DB 활용은 홍보 수신동의가 확인된 경우에만 추천한다.
- Action Score는 내부 검증용이며 사장님 화면에 직접 노출하지 않는다.
- 민감한 매출·광고 데이터를 URL에 넣지 않는다.
- `dist/`를 직접 수정하지 않는다. 원본을 수정한 뒤 `npm run build`로 생성한다.

## 주요 파일

- `src/engine.js`: 계산과 추천 규칙
- `src/ui-logic.js`: 입력 검증과 데이터 변환
- `src/app.js`: 화면 동작과 브라우저 저장
- `index.html`: 화면 구조와 문구
- `assets/styles.css`: 반응형 디자인
- `tests/`: 회귀·균형·화면 흐름 검증
- `docs/validation-plan.md`: 실제 고객 검증 기준

## 작업 전후 명령

```bash
npm ci
npm run verify
```

추천 규칙을 변경할 때는 먼저 실패하는 테스트를 추가하고, 36개 균형 시나리오의 특정 행동 쏠림 기준을 유지한다.
