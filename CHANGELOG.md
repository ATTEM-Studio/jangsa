# 변경 기록

## 12.0.0 - 2026-07-12

- 외부 매장 준비도와 내부 성장 병목 분리
- 미확인 항목 제외 및 60% 확인 범위 기준 추가
- 근거 신뢰도를 포함한 행동 우선순위 추가
- 실제 회복 배점 기반 목표점수 추가
- 링크 입력, 수동 확인, 실패 대체 흐름 추가
- 결과 화면에 준비도, 목표점수, 카테고리, 병목, CTA 계층 추가
- 민감한 매출·광고 원본값을 제외한 안전 공유 문구 추가
- 7일 결과 확인과 v2 실행 이력 추가

## 11.0.0 - 2026-07-11

- 5단계·19개 입력을 상황별 3단계 입력으로 축소
- 사장님 고민 중심의 첫 질문 도입
- 행동별 실행 가능 조건과 추천 근거 표시 추가
- 데이터 부족 시 확인할 숫자 안내
- 객단가 예상 효과를 선택률 20~40% 범위로 보수 계산
- 단일 행동 결과와 3단계 실행 순서 제공
- 추천 수락·불일치 피드백과 결과 기록 추가
- 36개 균형 시나리오와 화면 통합 테스트 추가
- GitHub Actions CI와 Pages 자동 배포 구성
