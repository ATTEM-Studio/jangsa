# 장사네비게이션 ver 11.0 고객 검증 계획

## 검증 질문

1. 사장님이 도움 없이 3분 안에 진단을 완료할 수 있는가?
2. 결과를 10초 본 뒤 오늘 할 행동을 자신의 말로 설명할 수 있는가?
3. 추천한 행동을 실제로 실행하는가?
4. 실행 후 확인할 숫자를 기록하고 다시 사용하는가?

## 1차: 화면 이해도 테스트

대상: 외식업 자영업자 6~10명

진행:

1. 결과 화면을 10초간 보여준다.
2. 화면을 가리고 오늘 무엇을 해야 하는지 묻는다.
3. 왜 그 행동인지, 첫 단계는 무엇인지, 어떤 숫자를 확인해야 하는지 묻는다.
4. 질문자가 설명하거나 힌트를 주지 않는다.

통과 기준:

- 오늘 행동 이해 정확도 80% 이상
- 추천 이유 이해 정확도 70% 이상
- 첫 실행 단계 이해 정확도 80% 이상
- 확인 지표 이해 정확도 70% 이상

## 2차: 전체 사용성 테스트

대상: 외식업 자영업자 5명 이상

측정:

- 도움 없는 진단 완료율
- 완료 시간 중앙값
- 잘못 입력한 항목 수
- `잘 모르겠어요` 선택 비율
- 추천 수락·불일치 비율
- SEQ 7점 척도
- SUS 10문항 점수

통과 기준:

- 도움 없는 완료율 80% 이상
- 완료 시간 중앙값 3분 이내
- 필수 입력 오류율 10% 이하
- SEQ 5.5/7 이상
- SUS 75점 이상
- 추천 불일치 응답 20% 이하

## 3차: 추천엔진 전문가 검증

외식업 마케터 2명이 시스템 결과를 보지 않고 36개 상황의 1순위 행동을 선택한다.

통과 기준:

- 전문가와 시스템의 1순위 일치율 75% 이상
- 실행 불가능하거나 위험한 추천 0건
- 데이터 부족 상황에서 억지 추천 0건
- 특정 행동 추천 비중 40% 이하
- 두 전문가 간 Cohen's kappa 0.6 이상

## 4차: 실제 매장 파일럿

대상: 서로 다른 업종의 매장 3~5개

기간:

- 1주 기준 수치 기록
- 2주 추천 행동 실행과 결과 기록

핵심 지표:

- 추천 행동 실행률 60% 이상
- 3일·7일 결과 기록률 50% 이상
- 유용성 평균 4/5 이상
- 2회 이상 재사용 매장 50% 이상
- 추천 불일치 응답 20% 이하

행동별 확인 지표:

| 행동 | 확인 지표 |
| --- | --- |
| 광고 화면 개선 | 전화·길찾기·예약 |
| 객단가 개선 | 세트 선택률·평균 객단가 |
| 재방문 행동 | 발송 대비 재방문·혜택 사용 |
| 지역 검색 개선 | 전화·길찾기·예약 |
| 소형 크리에이터 | 저장·예약·전화 |

## 중단·수정 기준

- 입력 완료율이 80% 미만이면 질문을 더 줄인다.
- 추천 일치율이 75% 미만이면 AI 문장을 추가하지 않고 추천 규칙부터 수정한다.
- 행동 실행률이 60% 미만이면 행동 범위를 10~20분짜리로 더 축소한다.
- 재사용률이 50% 미만이면 매일 코치가 아닌 주간 점검 도구로 재정의한다.
